Política de Privacidade

Última atualização: junho de 2026. Esta política está em conformidade com a Lei Geral de Proteção de Dados (LGPD — Lei nº 13.709/2018).

1. Quem Somos

A GHK·CU Glow é uma empresa brasileira especializada em insumos para pesquisa científica. Você pode nos contatar em: GHKCUGLOW@protonmail.com.

2. Dados que Coletamos

Coletamos apenas os dados estritamente necessários para as finalidades descritas nesta política:

Dados fornecidos diretamente por você

  • Dados de identificação: nome completo
  • Dados de contato: e-mail, número de WhatsApp
  • Dados de entrega: endereço completo (CEP, rua, número, complemento, cidade, estado)
  • Dados da transação: itens adquiridos, valor, forma de pagamento

Dados coletados automaticamente

  • Dados de navegação: endereço IP (anonimizado), tipo de navegador, páginas acessadas — coletados apenas se houver ferramenta de analytics ativa, o que será sempre informado nesta política.

Não coletamos dados sensíveis (origem racial, saúde, biometria, dados financeiros completos) sem consentimento explícito.

3. Como Usamos seus Dados

  • Processamento de pedidos: confirmação, despacho e rastreamento.
  • Comunicação: envio de informações sobre seu pedido via WhatsApp ou e-mail.
  • Atendimento ao cliente: resposta a dúvidas, reclamações e solicitações.
  • Obrigações legais: cumprimento de exigências fiscais e regulatórias.

Não utilizamos seus dados para marketing sem seu consentimento prévio.

4. Compartilhamento de Dados

Seus dados não são vendidos a terceiros. Podemos compartilhá-los apenas com:

  • Transportadoras: para viabilizar a entrega (nome e endereço).
  • Processadores de pagamento: para confirmação da transação.
  • Autoridades públicas: quando exigido por lei ou ordem judicial.

5. Retenção de Dados

Mantemos seus dados pelo tempo necessário para cumprir as finalidades desta política e nossas obrigações legais. Dados de pedidos são mantidos por 5 anos para fins fiscais, conforme exige a legislação brasileira. Após esse período, os dados são eliminados ou anonimizados.

6. Seus Direitos (LGPD)

Você tem os seguintes direitos em relação aos seus dados pessoais:

  • Confirmação e acesso: saber quais dados temos sobre você e acessá-los.
  • Correção: solicitar a correção de dados incompletos, inexatos ou desatualizados.
  • Eliminação: solicitar a exclusão de dados tratados com base no seu consentimento.
  • Portabilidade: receber seus dados em formato estruturado e interoperável.
  • Revogação de consentimento: retirar consentimento a qualquer momento.
  • Oposição: se opor a tratamentos realizados sem seu consentimento.

Para exercer seus direitos, entre em contato: GHKCUGLOW@protonmail.com. Respondemos em até 15 dias úteis.

7. Segurança

Adotamos medidas técnicas e organizacionais adequadas para proteger seus dados contra acesso não autorizado, alteração, divulgação ou destruição. Utilizamos comunicações criptografadas (HTTPS) e acesso restrito aos dados de pedidos.

8. Cookies

Este site pode usar cookies essenciais para funcionamento básico. Não utilizamos cookies de rastreamento ou publicidade comportamental sem seu consentimento.

9. Alterações nesta Política

Esta política pode ser atualizada periodicamente. A data de atualização está sempre indicada no topo. Alterações relevantes serão comunicadas de forma adequada.
